Add Properties

Instructions

  1. Open the console in the browser and run the test sabioTester('addProperties'). You will now see the specs you need to test for. Use this function to perform tests.
  2. Create a function called sabio.page.addProperties that takes an object.
    • Check if the parameter is null and not of type object. If so, the function should return the text "Not an object or null".
    • If the parameter is an object the function should add the properties name, age, gender and certificates to this object. Property certificates should be an empty array and the rest of the properties should be set to null.
    • sabio.page.addProperties should return the object with the properties added.
  3. Create a function called sabio.page.addId that takes an object.
    • Check if the parameter is null and not of type object. If so, the function should return the text "Not an object or null".
    • If the parameter is an object the function should add the property id to this object but USING bracket notation. Set the property to null.
    • sabio.page.addId should return the object with the property added.
