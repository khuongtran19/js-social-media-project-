Remove Letters

Instructions

  1. Open the console in the browser and run the test sabioTester('removeA'). You will now see the specs you need to test for. Use this function to perform tests.
  2. Create a function called sabio.page.removeA that takes a string.
    • Check if the parameter is null and not of type string (make sure you check for empty strings too). If so, the function should return the text "Please enter a valid string".
    • If the parameter passes the previous conditions then remove all the letters 'a' from the string. ( hint: make sure you check lower and upper cases).
    • sabio.page.removeA should return the string without a's.
  3. Create a function called sabio.page.countA that takes a string.
    • Check if the parameter is null and not of type string (make sure you check for empty strings too). If so, the function should return the text "Please enter a valid string".
    • If the parameter passes the previous conditions then count all the letters 'a' from the string. ( hint: make sure you check lower and upper cases).
    • sabio.page.countA should return the number of a's in the string.
Examples:
  • If we pass the string "Ask me anything please" to sabio.page.removeA it should return: "sk me nything plese".
  • If we pass the string "Bats are cheap at Andys" to sabio.page.countA it should return: 5.
