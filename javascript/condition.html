Conditional Statements

Instructions

  1. Open the console in the browser and run the test sabioTester('condStatement'). You will now see the specs you need to test for. Use this function to perform tests.
  2. Create a function called sabio.page.isNumber that takes a parameter and checks if the type of this parameter is an integer. If the parameter is an integer the function should return true and if not return false.
  3. Write a function called sabio.page.greaterNumber that takes two integers
    • Check if the parameters are integers using the sabio.page.isNumber function. If the parameters are not integers then it should return the text 'Please enter integers'.
    • If the parameters are integers then return the greater number or if both are the same return that number.
  4. Create a function called sabio.page.letterGrade that takes an integer.
    • Check if the parameter is an integer using the sabio.page.isNumber function. If the parameters is not an integer or if the value is not in the range of (0-100) then it should return the text 'Please enter an integer score between 0-100'.
    • If the parameter passes the previous conditions it should take the number and depending on this number return to the proper letter grade "A","B","C","D","F".
    • sabio.page.letterGrade should return the letter grade.
  5. Create a function called sabio.page.pluralizeNoun that takes two parameters. First should be a noun and second a number.
    • It should make the noun lower case and return the pluralized form. Example: "5 cats" or "1 dog".
    • Make it handle the collective nouns person (should be people) and goose (should be geese).
Examples:
  • If we pass the numbers 5 and 7 to sabio.page.greaterNumber it should return: 7.
  • If we pass the number 80 to sabio.page.letterGrade it should return: B.
  • If we pass person and 3 to sabio.page.pluralizeNoun it should return: 3 people.
  • If we pass car and 1 to sabio.page.pluralizeNoun it should return: 1 car.
  • If we pass CAR and 20 to sabio.page.pluralizeNoun it should return: 20 cars.
