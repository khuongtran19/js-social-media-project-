Calculator

Instructions

  1. Open the console in the browser and run the test sabioTester('calculator'). You will now see the specs you need to test for. Use this function to perform tests.
  2. Create an object called calculator.
  3. Add a property called errorMsg. Set the value of the property errorMsg to the string 'Please enter 2 numbers'.
  4. Add: add a property called add. Set the value of the property to be a function.
    • This function should take two parameters. It should check if the parameters are numbers (remember to check for nulls, undefined, etc).
    • If the parameters are not numbers it should return the errorMsg.
    • If the parameters are numbers the add method should return the sum of both numbers.
  5. Subtract: add a property called subtract. Set the value of the property to be a function.
    • This function should take two parameters. It should check if the parameters are numbers (remember to check for nulls, undefined, etc).
    • If the parameters are not numbers it should return the errorMsg.
    • If the parameters are numbers the subtract method should return the subtraction of the first by the second parameter.
  6. Multiply: add a property called multiply. Set the value of the property to be a function.
    • This function should take two parameters. It should check if the parameters are numbers (remember to check for nulls, undefined, etc).
    • If the parameters are not numbers it should return the errorMsg.
    • If the parameters are numbers the multiply method should return the multiplication of both numbers.
  7. Divide: add a property called divide. Set the value of the property to be a function.
    • This function should take two parameters. It should check if the parameters are numbers (remember to check for nulls, undefined, etc).
    • If the parameters are not numbers it should return the errorMsg.
    • If the second parameter is 0 then it should return the message 'Divide by zero error'.
    • If the parameters are numbers the divide method should return the division of the first by the second.
Examples:
  • If we pass the values 2.1 and 4.1 to calculator.add it should return: 6.2.
  • If we pass the values 5 and 2.5 to calculator.subtract it should return: 2.5.
  • If we pass the values 10 and 3 to calculator.multiply it should return: 30.
  • If we pass the values 20 and 2 to calculator.divide it should return: 10.
  • If we pass the values 30 and "hi" to any method it should return: 'Please enter 2 numbers'.
